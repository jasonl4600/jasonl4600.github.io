CYOA-Project

In this project we were given the task to create a storyline on github based on the prompt we pick. Before this we all as a class created multiple prompts each and then later on had to pick other peoples prompts or our own that we created. We then partner up with the person that chose the same prompt as us to work together in which my case I was working with Leonel. We had to create a plan/drawing for this before starting at all to map out the actual story of this. In our project we created a storyline about gaining the powers of the character in the show we watch.

Surprisingly the biggest challenge I had was mapping out what we were making. At first it seemed really simple just make up a story with our prompt and go from there making me think creating the story on cs50 would be the hard part. But I was wrong because actually mapping out the causes and branches of the story would be harder than creating it. This goes to show that planning is a very important process in creating something which is what I took away during this project.
