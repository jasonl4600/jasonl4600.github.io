Sep11 Freedom Project

Context

The Freedom Project is a year long project in the class of SEP11 Javascript where we use the Javascript we learn all year long in order to create a year long project. The freedom project in Sep11 is a year long project that requires you to pick a tool and create something using Javascript with that tool as your project for the whole year. This meant on the process of learning Javascript in class we will also be learning our own tool.

Content

For my project, I used the tool KaboomJs. KaboomJs is a javascript library that allows the user to easily create games with the usage of Javascript. Using Kaboom, I made a platformer game similar to ones you may know wlike Mario.< Kaboom is made to be easily learned and taught in order to create games./p>

Reflection

Throughout the process of learning my tool KaboomJs while learning Javascript in general in class, I've faced many challenges. Some of these challenges include the struggling of being able to incorporate key elements of what you're learning in class and the tool you are using. Learning your tool in general can be a challenge. In my case Kaboom didn't have the greatest documentation. I persevered by findign more resources that I could learn from in order to get my project where it is today.

Takeaways

  • Always read Documentation, but don't forget about many other resources (Use Google).
  • Ask for help (Slack or Mr. Mueller or Friends)
  • Take your time (Learn your tool the best way you can to understand it better)
  • Trust yourself (Even if something seems too hard for you, don't give up.)
